Why Twine?

Write UCM services, Idoc script extensions and filters using POJOs.

public class ExampleScriptPackage {

  @IdocFunction
  public int strLength(String input) {
    return input.length();
  }

  // usage: <!--$ strLength("Hello") -->
  // result: 5
}

Stop writing all that same parameter checking code:

public class ExampleServicePackage {

  @ServiceMethod(name = "EXAMPLE_SERVICE")
  public void example(@Binder(name = "date") Date date) {
    // do something with date
  }

  // usage: ?IdcService=EXAMPLE_SERVICE&d=Hello
  // result: Error: Parameter 'date' must be parseable as a date
}
